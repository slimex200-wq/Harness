# PRD: DailyLoop Web - 습관 추적 웹 앱

## 개요
- **한 줄 설명**: 매일 반복하는 습관을 체크하고 연속 달성(스트릭)을 시각화하는 미니멀 웹 앱
- **대상 사용자**: 자기 관리에 관심 있는 20-40대
- **핵심 가치**: "오늘 할 일을 체크하는 3초"

## 기술 스택

| 계층 | 기술 |
|------|------|
| Framework | Next.js 14+ (App Router) |
| Language | TypeScript (strict) |
| Styling | Tailwind CSS |
| Database | Prisma + SQLite |
| Auth | 없음 (로컬 전용, MVP) |
| Testing | Vitest + React Testing Library |

## 데이터 모델 (Prisma)

```prisma
model Habit {
  id          String   @id @default(cuid())
  name        String
  icon        String   @default("check")
  color       String   @default("#6366f1")
  frequency   String   @default("daily") // daily | weekdays | custom
  customDays  String   @default("[]")    // JSON array [0-6]
  order       Int      @default(0)
  archived    Boolean  @default(false)
  createdAt   DateTime @default(now())
  checks      Check[]
}

model Check {
  id        String   @id @default(cuid())
  date      String   // "YYYY-MM-DD"
  habitId   String
  habit     Habit    @relation(fields: [habitId], references: [id])
  createdAt DateTime @default(now())

  @@unique([date, habitId])
}
```

## 기능 (MVP Only)

| ID | 기능 | 수용 기준 |
|----|------|----------|
| M1 | 습관 생성 | 이름 필수, 아이콘/색상/반복주기 선택. 최대 20개 |
| M2 | 오늘의 체크리스트 | 홈에 오늘 활성 습관 표시. 클릭으로 완료 토글 |
| M3 | 스트릭 표시 | 각 습관 옆 연속 달성 일수. 하루 빠지면 0 리셋 |
| M4 | 주간 캘린더 뷰 | 상단 7일 도트 그리드. 완료일 색상 채움 |
| M5 | 습관 편집/삭제 | 이름/설정 변경, soft delete (archive) |
| M6 | 다크 모드 | 시스템 설정 따르기 + 수동 전환 |
