## Model Routing Policy

| 태스크 유형 | 모델 | 에이전트/스킬 |

| 코드 생성/수정 | sonnet | build-error-resolver |
| 코드 리뷰/검증 (read-only) | haiku | flutter-review, flutter-verify |
| 아키텍처 분석 (read-only) | haiku | flutter-arch, planner |
| 보안 스캔 (read-only) | haiku | firebase-security |
| eval 실행 (code grader) | haiku | flutter-eval (code 부분) |
| eval 실행 (model grader) | haiku | flutter-eval (model 부분) |
